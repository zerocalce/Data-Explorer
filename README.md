### Technical Analysis & Specification: Data-Explorer System
The Data-Explorer is a high-performance, full-stack application designed for Scientific Data Visualization with a focus on organometallic coordination polymers. It leverages a modern "Cyberpunk" aesthetic to provide an immersive, high-visual-impact dashboard experience.

### 1. Core Technology Stack
Layer Technology Role Frontend React 19 Core UI library for component-based architecture. Styling Tailwind CSS 4.0 Utility-first CSS for custom cyberpunk design and layouts. Animations Framer Motion High-fidelity animations, glitch effects, and transitions. UI Kit Shadcn UI Accessible, consistent components (Progress, ScrollArea, Tabs). Backend Express 5.0 RESTful API and server-side logic orchestration. Database PostgreSQL + Drizzle Type-safe persistence layer (current impl. in-memory). Validation Zod Runtime type checking for API inputs/outputs.
